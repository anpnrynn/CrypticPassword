# Inspiration:       

1. Pin + Pattern gives improved security.     
2. Pattern in Google's Android phone can only be drawn in one way. In this scenario, even if you look at the pattern, there could be multiple combinations of it.
      1. 1, 2, 3 Cell numbers, could be drawn as 1,2,3 or 2,1,3 or 3,2,1 ... etc (Improving security)
      2. Also this kind of pattern can be used for Desktops, Laptops and Computers, not just personal devices.
3. 2 Algorithms + 2 different Seed values gives for 4 variations of the Password with the same Pin and Pattern.         
 
# Algorithm Related Inspiration:       
1. Instead of using bitwise operators, it uses character swaps and rotations by making use of random numbers (shuffling of characters).
2. Bitwise operation causes repeats
3. Also, this is clear text, so bitwise operators aren't very useful.
